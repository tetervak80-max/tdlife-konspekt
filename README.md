# TDLife Konspekt — Скіл для Claude Code

Скіл автоматично генерує **двомовний інтерактивний HTML-конспект** з будь-якого транскрипту уроку, вебінару або виступу.

## Що він робить

- Отримує транскрипт → одразу генерує готовий HTML-файл
- **Двомовний**: українська + російська, перемикач мов вбудований
- Вибір мови зберігається в localStorage
- Акордеон-секції (розкриваються/закриваються)
- Інтерактивний чек-лист дій
- Фірмовий стиль **TDLife** (кольори, шрифти, картки)
- Футер з контактами автора
- Mobile-friendly

## Як встановити

1. Склонуй репо або завантаж `SKILL.md`
2. Скопіюй папку в `~/.claude/skills/tdlife-konspekt/`
3. Перезапусти Claude Code — скіл з'явиться автоматично

```bash
mkdir -p ~/.claude/skills/tdlife-konspekt
curl -o ~/.claude/skills/tdlife-konspekt/SKILL.md \
  https://raw.githubusercontent.com/tetervak80-max/tdlife-konspekt/main/SKILL.md
```

## Як використовувати

Просто вставте транскрипт в чат — скіл спрацює автоматично без додаткових команд.
